# Orchestration Skills

5 bundled skills.

## `decision-facilitation`

Turn design disagreement or ambiguity into explicit criteria, trade-offs, reversible decisions, and proportionate tests.

## `design-critique`

Critique a product experience by intent, user outcome, structure, interaction, system, and craft, then prioritize actionable corrections.

## `product-design-orchestrator`

Coordinate a product-design effort from uncertain request through evidence, design, implementation, verification, and live learning without applying unnecessary ceremony.

## `project-inspector`

Inspect an existing product and repository before UI changes so design decisions extend the real system instead of inventing a parallel one.

## `skill-router`

Select the smallest useful bundled or external skill set by decision, lifecycle phase, risk, platform, and specificity.
