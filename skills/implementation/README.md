# Implementation Skills

10 bundled skills.

## `design-system-integration`

Integrate new UI into an existing design system by reusing primitives, adding justified tokens/components, and preventing local drift.

## `design-to-code`

Translate approved flows, screen contracts, tokens, components, content, and states into maintainable production code in vertical slices.

## `i18n-content-resilience`

Harden UI for localization, RTL, text expansion, locale formats, input methods, pluralization, and extreme/missing content.

## `metadata-discoverability`

Implement accurate page/app metadata, structured sharing, indexing controls, deep links, titles, and install/discovery surfaces without misleading users or crawlers.

## `native-ui-engineering`

Implement faithful, accessible, performant native or cross-platform mobile/desktop UI using platform navigation, layout, input, lifecycle, and accessibility conventions.

## `performance`

Design, implement, and verify UI performance for loading, responsiveness, visual stability, animation, memory, and realistic low-end conditions.

## `pixel-perfect-implementation`

Recreate an approved visual reference with measured, maintainable, responsive implementation and a deterministic screenshot comparison loop.

## `stack-inspector`

Detect a project’s platform, framework, versions, rendering, styling, state, data, component, test, and deployment conventions before UI implementation.

## `technology-adapter`

Translate approved design behavior into the conventions and capabilities of any detected technology stack using current official documentation.

## `web-ui-engineering`

Implement semantic, responsive, accessible, performant web UI with robust browser behavior, intrinsic layout, and maintainable component boundaries.
